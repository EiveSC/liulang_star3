# 🌹 流浪写真集 - 外网访问

## 🌐 外网访问地址

**ngrok 链接**：请查看 http://localhost:4040

或执行命令获取：
```bash
curl -s http://localhost:4040/api/tunnels | python3 -m json.tool
```

## 📍 本地访问地址

```
http://localhost:8080/
```

## 🚀 启动外网穿透

```bash
cd /home/admin/openclaw/workspace/photo-gallery

# 1. 启动本地服务器
python3 -m http.server 8080 &

# 2. 启动 ngrok 隧道
ngrok http 8080
```

## 📊 查看访问统计

```
http://localhost:4040
```

## 📱 访问方式

1. 从 ngrok 管理界面获取外网链接
2. 复制链接到微信或浏览器
3. 点击访问流浪写真集
